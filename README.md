# Apple iPhone Sales Analysis

A data analysis project that explores Apple iPhone product data using Python, Pandas, NumPy, and Plotly. The objective of this project is to perform Exploratory Data Analysis (EDA), identify customer preferences, analyze pricing strategies, and derive business insights from ratings, reviews, discounts, and product pricing data.

---

## Project Objective

The goal of this project is to:

- Analyze iPhone pricing and discount trends.
- Identify the most popular and highly rated iPhone models.
- Understand the relationship between ratings, reviews, discounts, and sale prices.
- Visualize key insights using interactive charts and data visualizations.
- Apply real-world data analysis techniques using Python.

---

## Dataset Information

The dataset contains Apple iPhone product information, including:

- Product Name
- Sale Price
- MRP (Maximum Retail Price)
- Discount Percentage
- Number of Ratings
- Number of Reviews
- Star Rating
- Product URL

---

## Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Statsmodels

## Key Insights

- Premium iPhone models received the highest customer engagement in terms of ratings and reviews.
- Customer ratings remained relatively consistent across different price ranges.
- Discounts showed varying influence on product pricing but did not significantly affect customer ratings.
- Products with higher review counts generally demonstrated stronger market popularity.
